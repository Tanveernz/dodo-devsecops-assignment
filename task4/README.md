# Task 4 – Security Assessment

## Objective

The objective of this task was to perform a structured security assessment while respecting the engagement scope provided in the assignment.

The assessment was divided into two phases:

1. Passive Reconnaissance
2. Penetration Testing of the authorized target

---

## Scope

### In Scope

- Passive OSINT against the public target
- Security assessment of the provided vulnerable application
- Local testing environment

### Out of Scope

- Production exploitation
- Denial of Service
- Credential attacks
- Brute force attacks
- Social engineering

---

## Deliverables

- Reconnaissance Report
- Penetration Test Report
- Risk Assessment
- Security Recommendations

---

## Methodology

The assessment followed a standard penetration testing methodology:

- Information Gathering
- Enumeration
- Vulnerability Identification
- Risk Assessment
- Reporting

---

## Conclusion

The assessment focused on identifying common security weaknesses and documenting practical remediation steps while remaining within the authorized testing scope.
